# Movement Speed
https://predecessor.wiki.gg/wiki/Movement_Speed

## Overview

Movement Speed (MS) is a stat present in every Hero in Predecessor, each with varying base and scaling values. This attribute can be enhanced through leveling up heroes, utilizing certain crest actives like Soulbearer's Anima effect, or equipping specific items such as Berserker's Axe.

## Movement Penalties

The game implements several penalties to movement speed:

### Attack Penalties
- Melee heroes: 25% penalty while attacking
- Ranged heroes: 30% penalty while attacking
- Melee heroes receive an additional 5% penalty when missing an attack

### Directional Penalty
- All heroes experience a 15% movement speed reduction when moving sideways

## Items Providing Movement Speed Bonuses

The following equipment pieces grant movement speed increases:

- Berserker's Axe
- Dust Devil
- Painweaver
- Solstone Drone
- Void Conduit
- Windcaller
- Wraith Leggings
