# Mana Regen
https://predecessor.wiki.gg/wiki/Mana_Regen

## Overview

There are two types of Mana Regen gains in Predecessor:

| Type | How to Gain | Description |
|------|-----------|-------------|
| Base Mana Regen | Items | Increases a hero's base mana regen value by a percentage |
| Max Mana Regen | Blue Buff, River Buffs, and Orb Prime | Adds a flat bonus to the mana regen value |
